Swap Source

Terraform

1# @swapsource(remote_source="", remote_version="", local_source="")
2module {
3   source = ""
4}

1# @swapsource(remote_source="", remote_version="", local_source="")
2module {
3   source = ""
4   version = ""
5}

usage: tfutility sourceswap [-h] [-s {l,r,local,remote}] paths [paths ...]

Positional Arguments

paths

Path to one or more TF-Files. Its also Possible to set a folder. It searches only *.tf files

Named Arguments

-s, --switch-to

Possible choices: l, r, local, remote

Define to which reference all decorated modules should be swapped
